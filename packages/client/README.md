# @stakit/client

TypeScript client for [stakit](../../) routers. (Scaffold — implementation next.)

```bash
bun install          # from repo root
bun --filter '@stakit/client' run build
```

`bun run build` produces:

```
dist/esm/    # ES modules  (.js)
dist/cjs/    # CommonJS    (.cjs)
dist/types/  # declarations (.d.ts)
```

Scripts: `build`, `typecheck`, `lint`, `lint:fix`, `format`, `format:fix`,
`test`, `code-check`.
