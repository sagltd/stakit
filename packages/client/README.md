# @stakit/client

Typed TypeScript client for [stakit](../../) routers ‚Äî HTTP (unary + stream) and
WebSocket (duplex). Generic over the `Router` type your server emits via
`router.generate_ts()`. Never throws on an application error (those ride in the
result and are narrowed with `isOk` / `isError`); only real network failures
reject. See `docs/transport.md` for the wire contract.

## Create a client

```ts
import { createClient, isOk, isError } from '@stakit/client'
import type { Router } from './types' // generated by stakit-router

const client = createClient<Router>({
  url: 'https://api.example.com/app', // base (used for every call)
  headers: { authorization: token }, // base headers
  streamUrl: 'https://api.example.com/stream', // optional (defaults to url)
  wsUrl: 'wss://api.example.com/ws', // optional (defaults to url)
  defineClientActions: {
    // server‚Üíclient handlers
    showToast: (p) => {
      /* ‚Ä¶ */ return 'ok'
    },
  },
  onError: (err) => err, // fires only on a real network failure
})
```

## Unary ‚Äî one or many actions in one request

```ts
const r = await client.fetch({
  greet: { name: 'alice' },
  version: null,
})

if (isOk(r.greet)) r.greet.data.message // typed
if (isError(r.greet)) r.greet.error.code // { code, message, fields? }
```

`fetch` takes a map of `action ‚Üí params` and returns a map of
`action ‚Üí ActionResult`. `isOk` / `isError` narrow each entry to `.data` /
`.error`.

## Per-call overrides (fan out to any server)

Every field is optional and affects only that call ‚Äî the client base is never
mutated.

```ts
await client.fetch(
  { greet: { name: 'x' } },
  {
    url: 'https://vm-42.internal/app', // override base url
    headers: { authorization: vmToken }, // merged over base headers
    files: [file], // multipart upload
  },
)
```

## Stream

```ts
for await (const frame of client.stream({ count: { n: 5 } })) {
  if (isOk(frame.count)) console.log(frame.count.data)
}
```

## WebSocket (duplex + server‚Üíclient calls)

```ts
const conn = client.connect({
  defineClientActions: {
    showToast: (p) => {
      /* ‚Ä¶ */ return 'ok'
    },
  },
})
await conn.send({ progress: { n: 3 } })
for await (const frame of conn.stream) {
  // server‚Üíclient `client_call`s are auto-dispatched to defineClientActions
}
conn.close()
```

## Headers

```ts
client.setHeaders({ authorization: t }) // replace all
client.setHeaders((prev) => ({ ...prev, authorization: t })) // functional update
```
